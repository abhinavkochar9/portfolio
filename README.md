# Portfolio

Personal portfolio website showcasing projects, research, skills, and experience as a Data Scientist & AI/ML Engineer.

## Features

- Dark/light theme toggle
- Responsive project and skills grid
- Interactive sidebar navigation
- Contact form
- Hosted on Streamlit Cloud

## Tech Stack

- **Streamlit** — Web framework
- **Plotly** — Interactive visualizations
- **Pandas / NumPy** — Data handling

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Structure

```
├── app.py              # Main application (pages, styling, content)
└── requirements.txt    # Dependencies
```

Customize content by editing the data dictionaries at the top of `app.py`.
